# TaNaMesa

Sistema web de comanda para bar/restaurante, desenvolvido para portfólio e aprendizado. O projeto simula o fluxo de atendimento de um estabelecimento, permitindo que o cliente faça pedidos pelo site, enquanto cozinha, caixa e gerente acompanham e gerenciam as operações.

## Objetivo do projeto

O objetivo do TaNaMesa é centralizar o processo de pedidos e controle de mesas em um único sistema, tornando o atendimento mais organizado, rápido e prático.

## Funcionalidades previstas

- Seleção de itens do cardápio pelo cliente
- Envio dos pedidos para a cozinha
- Visualização dos pedidos pelo caixa
- Atualização automática da conta da mesa
- Gerenciamento de mesas
- Adição e remoção de itens do cardápio pelo gerente

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Python
- MySQL

## Estrutura do sistema

O sistema foi pensado para atender diferentes áreas do estabelecimento:

### Cliente
- Visualiza o cardápio
- Escolhe os itens
- Realiza pedidos pelo site

### Cozinha
- Recebe os pedidos realizados
- Visualiza mesa, nome do cliente e item solicitado

### Caixa
- Acompanha os pedidos
- Visualiza o valor atualizado da conta

### Gerente
- Libera mesas
- Gerencia os itens do cardápio
- Controla o fluxo operacional
